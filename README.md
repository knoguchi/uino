# Uino (初乃)

Predictive coding cortex driven by spike streams from peripheral sensory crates:

- Audio: [cochlea-rs](https://github.com/knoguchi/cochlea-rs)
- Vision: [retinula](https://github.com/knoguchi/retinula)

Design plan: [uino_再設計計画書.md](uino_再設計計画書.md).

## Build

```bash
cargo build
cargo test
```
